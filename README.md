# AI Powered Text-to-SQL Assistant

An AI-powered application that converts natural language questions into SQL queries using **LangChain**, **Groq LLM**, **Streamlit**, and **MySQL**.

Users can ask database-related questions in plain English, and the application automatically:
- Generates SQL queries
- Executes them on MySQL
- Displays the results instantly

---

## 🚀 Features

- Convert natural language to SQL queries
- Connects directly with MySQL database
- Uses Groq LLM (`llama-3.3-70b-versatile`)
- Interactive Streamlit web interface
- Automatic query execution
- Displays generated SQL and query results

---

## 🛠 Technologies Used

- Python
- Streamlit
- LangChain
- Groq API
- MySQL
- SQLAlchemy
- PyMySQL
